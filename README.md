# 📚 Study Tracker App

React & Node.js を用いた学習記録アプリケーションです。勉強時間を日・週・月ごとに記録・集計・ランキング表示ができます。

---

## 🔧 使用技術

| 種別 | 技術 |
|------|------|
| フロントエンド | React / Bootstrap / Axios |
| バックエンド | Express / MongoDB Atlas / JWT |
| 認証 | JSON Web Token (JWT) |
| デプロイ | Render |

---

## 🚀 セットアップ手順（ローカル）

```bash
# frontend
cd frontend
npm install
npm start

# backend
cd backend
npm install
npm run dev
